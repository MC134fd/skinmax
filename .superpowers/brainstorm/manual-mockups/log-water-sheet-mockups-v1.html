Color Tokens

  • Sky Blue — hero hydration accent
  • Sky Deep — droplet gradient end
  • Hero Coral — primary Log CTA only
  • Cream Base — sheet background
  • Soft Tan — grabber, dividers

Auto-Locale Units

  • US locale → default oz. Presets 8 / 16 / 24 oz. Goal 64 oz.
  • Everywhere else → default ml. Presets 250 / 500 / 750 ml. Goal 2,000 ml.
  • User can override via sheet header chip (persists to settings).
  • Internally stored in ml; oz is a display conversion.

Haptics

  • .light on preset tap / pour add
  • .selection on unit toggle
  • .success on Log commit
  • .warning on Undo pour

Typography

  • Total / hero number → .gbDisplayL
  • Sheet title → .gbTitleM
  • Preset name → .gbCaption
  • Preset amount → .gbOverline tracking 1.4
  • CTA → .gbTitleM
